# openclaw-skill

A minimal Python scaffold for an OpenClaw skill.

Quick start

- Create a virtualenv: `python -m venv .venv`
- Activate it: `source .venv/bin/activate`
- Install deps: `pip install -e .`
- Run: `python -m openclaw_skill`
